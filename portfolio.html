ผลงาน

การเรียนรู้เขียนโปรแกรมเบื้องต้น

ดิฉันได้เริ่มเรียนรู้การเขียนโปรแกรมเบื้องต้นด้วยภาษา Python จากแหล่งเรียนออนไลน์ เช่น Codecademy และ YouTube โดยฝึกเขียนโปรแกรมง่าย ๆ เช่น เกมเป่ายิ้งฉุบ เครื่องคิดเลข และโปรแกรมสุ่มคำศัพท์ ซึ่งช่วยให้เข้าใจพื้นฐานของการคิดเชิงตรรกะและโครงสร้างของภาษาโปรแกรม

ดูรายละเอียด

การเข้าร่วมค่ายหรือกิจกรรมด้านเทคโนโลยี

เคยเข้าร่วมค่ายเขียนโปรแกรมที่จัดโดยหน่วยงานการศึกษา ซึ่งได้เรียนรู้การพัฒนาโปรแกรมด้วย Scratch และการสร้างเกมพื้นฐาน ส่งผลให้เกิดความสนุกและความท้าทายในการแก้ไขปัญหา พร้อมทั้งได้ฝึกการทำงานร่วมกับผู้อื่นในรูปแบบของทีม

ดูรายละเอียด

การทำโปรเจกต์ส่วนตัว

เพื่อฝึกฝนทักษะการคิดและการแก้ไขปัญหา ดิฉันได้ลองพัฒนาเว็บไซต์แนะนำตัวเองด้วยภาษา HTML และ CSS โดยศึกษาและทดลองจากคู่มือออนไลน์ การได้เห็นผลงานของตนเองสามารถแสดงผลบนหน้าเว็บจริง ๆ ทำให้เกิดแรงบันดาลใจอยากเรียนรู้เพิ่มเติมในสายงานนี้
